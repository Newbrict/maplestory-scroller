# maplestory-scroller
Maplestory scroll probability / cost calculator

```
usage: scroller.py [-h] -i INPUT_EQUIP -o OUTPUT_EQUIP SCROLL_DIRECTORY

Maplestory scroll cruncher

positional arguments:
  SCROLL_DIRECTORY      The directory containing all scroll json files for
                        this crunch

optional arguments:
  -h, --help            show this help message and exit
  -i INPUT_EQUIP, --input-equip INPUT_EQUIP
                        The json file representing the equip you are starting
                        with
  -o OUTPUT_EQUIP, --output-equip OUTPUT_EQUIP
                        The json file representing the equip you want to end
                        up with
```
